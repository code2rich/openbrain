# 读书方法论

## 阅读策略

| 方法 | 特点 | 适用场景 |
|------|------|---------|
| 问题导向法 | 带着问题读书 | 解决具体问题 |
| 两遍法（叔本华） | 第一遍通读、第二遍精读 | 经典著作 |
| 精读+泛读结合 | 核心精读，其余泛读 | 专业书籍 |
| 定向选读 | 只读相关章节 | 工具书/参考书 |

## 读书方法分类

### 过程实操

- 诵读与默读结合
- 笔记批注法（随手批注/便签/摘抄）
- 研读难点法

### 记忆与输出

- 5 分钟复盘 + 隔天回顾
- 讲授法（费曼技巧）
- 思维导图
- AI 辅助阅读法（2026 新趋势）

### 高效时间分配

DeepSeek 推荐：30% 阅读 + 70% 预读/输出

## 知行合一五步法

1. **筛选触动点**：拒绝贪多求全，单点突破
2. **关联现实场景**：书写行动清单（具体场景 + 具体动作）
3. **知行迭代**：小步快跑，试错-复盘-修正循环
4. **深度内化**：从"读故事"到"读自己"（以书为镜叩问内心）
5. **输出倒逼输入**：以教为学，分享/教授检验理解

## 快速思考与表达

### 结构框架

- **三段式**：开场 - 主体 - 结语
- **SCQA 模型**：情境 - 冲突 - 问题 - 答案
- **直指本质式**：一针见血

### 思维工具

- 逻辑推演：归纳 / 演绎 / 对照
- 生动论证：举例 / 比喻 / 引述
- 关键词记忆法

### 日常修养

"站着思考"习惯，"五多"训练：多读书 / 多思考 / 多练习 / 多观摩 / 多演练
