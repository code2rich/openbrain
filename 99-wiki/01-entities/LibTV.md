# LibTV

LiblibAI 推出的一站式 AI 视频创作平台，定位不是单纯的生成入口，而是将整套视频创作流程组织在统一画布中完成。

## 基本信息

- **所属公司**：LiblibAI
- **海外品牌**：Lovart
- **官网**：https://www.liblib.tv/
- **Skill 仓库**：https://github.com/libtv-labs/libtv-skills

## 核心特性

### 节点式工作流 + 无限画布

采用节点编辑器 + 无限画布的创作方式，剧本、分镜、人物设定、参考图、视频节点、剪辑节点全部放在同一项目中统一管理。核心解决的问题：素材散、角色散、镜头散、思路散。

### 已接入模型

- **Seedance 2.0**（即梦视频生成模型）— 满血版，无降智
- **NanoPro**（香蕉模型）— 图片处理，如去背景
- **可灵 O3 + 可灵 3.0** — 订阅用户赠送共 300 条免费最高等级视频权益

### 20+ 专业视频创作功能

- 9/25 宫格分镜生成
- 一键生成完整故事板
- 剧情推演四宫格
- 多机位 9 宫格镜头设计
- 角色三视图生成
- 视频拆解（反向工程高质量视频的镜头、节奏、结构）
- 一键融图（电商换装）
- 一键复用预设模板

### Agent/Skill 驱动

LibTV 不只服务手动创作者，也在为 Agent 预留能力入口。已开放 Skill 包括：

- 短漫剧生成 Skill
- 爆款视频复刻 Skill
- 音乐 MV 生成 Skill

Personal Agent（如"小龙虾"）可直接调用 LibTV 创作能力，实现写剧本 → 拆分镜 → 出片 → 剪辑的自动化流程。

## 适用场景

| 场景 | 说明 |
|------|------|
| 产品宣传视频 | 上传产品图 → 去背景 → 提示词生成宣传视频 |
| 电商种草视频 | 一键融图换装 → 场景切换 → 生成平台适配视频 |
| 微电影/连续剧情 | 画布统一管理角色、场景资产，保持一致性 |
| MV / 漫剧 | Skill 驱动，一键生成 |

## 定价优势（vs 即梦）

- 年卡最低 39 折
- 部分模型额外 6 折优惠（综合约 2 折多）
- 会员 SKU 价格比竞品低 76%
- 模型积分定价比竞品低 92%
- 无排队问题

## 来源观点

- LibTV 接入的 Seedance 2.0 与即梦官方完全一致，不降智
- 画布工作流的最大优势是资产统一管理，创作思路不轻易断裂
- 决定效率的已不只是模型本身，而是整套创作流程能不能跑顺

## insights

- LibTV 的画布模式本质上是把「视频创作」从单点生成升级为项目管理，这对需要持续产出系列内容的创作者很有价值
- Agent Skill 接口是一个值得关注的方向——当 AI Agent 能直接调用视频创作能力时，内容生产的自动化程度将大幅提升
