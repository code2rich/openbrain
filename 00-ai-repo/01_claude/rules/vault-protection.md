# Vault 保护规则

## 三层分离原则

```
00-raw/    → 只读，AI 永不修改
99-wiki/   → AI 完全拥有，持续进化
100-output/ → 按需生成
```

## 不可变真相层 (00-raw/)

- AI 只读取，不修改、不删除、不移动原始资料
- 新的原始资料来源统一放入 `00-raw/`，命名采用数字序号前缀
- raw 文件保持原始来源的命名 + 时间戳

## 结构化知识层 (99-wiki/)

- 每次交互都在完善知识体系（复利效应）
- 页面必须区分「来源观点」和「个人洞察」
- 周期/临时知识必须设置 `review_cycle`
- 知识被使用时记录 `used_in`，未使用的知识是负债

## wikiapp 子系统

- 不要修改 99-wiki 的原始文件 — wikiapp 只读取
- 不要手动编辑 `.vitepress/generated/` — 全部自动生成
- `wiki/` 是 symlink — 指向 `.wiki-preprocessed/`
